List of Command Keywords in TankTask:

add
Format: add [Task Name]
Example: add Feed the fish

delete (Alternate form: del)
Format 1: del [Task Index]
Example: del 1
Format 2: [Task Index] del
Example: 1 del

done
Format 1: done [Task Index]
Example: done 1
Format 2: [Task Index] done
Example: 1 done

undone
Format 1: undone [Task Index]
Example: undone 1
Format 2: [Task Index] undone
Example: 1 undone

undo
Format: undo

search
Format: search [Search Term(s)]
Example: search fish

deadline (Alternate form: by, dea)
Format: [Task Index] by [Deadline]
Example: 1 by 20 Oct 3pm
Example: 1 by tmr
Example: 1 by next tues

event (Alternate form: from)
Format: [Task Index] from [Start] to [End]
Example: 1 from 20 Oct 3pm
Example: 1 from 20 Oct 3pm to 5pm
Example: 1 from 20 Oct 3pm to 21 Oct 8am

description (Alternate form: des)
Format: [Task Index] des [Description]
Example: 1 des need to do this

priority (Alternate form: pri)
Format: [Task Index] pri [Priority (1-5)]
Example: 1 pri 5

reminder (Alternate form: rem)
Format: [Task Index] rem [Reminder Date]
Example: 1 rem 20 Oct 3pm

category (Alternate form: cat)
Format: [Task Index] cat [Category Name]
Example: 1 cat Hobbies

rename
Format: [Task Index] rename [New Task Name]
Example: 1 rename Kill the fish

reset
Format: [Task Index] reset [Field (all/dea/event/des/pri/rem/cat)]
Example: 1 reset all
Example: 1 reset des

every
Format: every [Frequency (day/week/month/year)] [Date/Time]
Example: every day 3pm
Example: every week fri 11am
Example: every month 15th
Example: every year Apr 1

show (Alternate form: showT, showE, showF)
Format 1: show [Task Type (Todo/Event/Floating)]
Example: show event
Format 2: show[X]
Example: showE

sort (Alternate form: sortD, sortN, sortP)
Format 1: sort [Field (Deadline/Name/Priority)]
Example: sort priority
Format 2: sort[X]
Example: sortP

exit
Format: exit


One-shot Command Examples:

Add ToDo
Format: add [Task Name] by [Deadline]
Example: add Feed the fish by today

Add Event
Format: add [Task Name] from [Start] to [End]
Example: add Feed the fish from today 6pm to 6:15pm

Set/Edit Information
Format [Task Index] [command1] [command1's content] [command 2] [command2's content]...
(Commands available in this mode are deadline/event, description, priority, reminder, category, rename)
Example: 1 pri 2 rem tomorrow 12pm cat hobbies des running out of fish food
