List of Command Keywords in TankTask:
Adding Tasks

add
	Format: add [Task Name]
		Example: add Feed the fish
		
Add ToDo
	Format: add [Task Name] by [Deadline]
		Example: add Feed the fish by today

Add Event
	Format: add [Task Name] from [Start] to [End]
		Example: add Feed the fish from today 6pm to 6:15pm
break

List of Command Keywords in TankTask:

One-shot Command Examples:

Add ToDo (with description and priority)
	Format: add [Task Name] by [Deadline] des [Description] pri [Priority]
		Example: add Feed the fish by today des slightly less than the usual amount pri 4

Add Event (with reminder)
	Format: add [Task Name] from [Start] to [End] rem [Reminder]
		Example: add Feed the fish from today 6pm to 6:15pm rem 6pm today

Set/Edit Information
Format [Task Index] [command1] [command1's content] [command 2] [command2's content]...
(Commands available are deadline/event, description, priority, reminder, category, rename)
	Example: 1 pri 2 rem tomorrow 12pm cat hobbies des running out of fish food 
break
